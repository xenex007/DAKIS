# DAKIS 🦊🤖

**DAKIS** est une interface web moderne, ultra-rapide et autonome (single-file) conçue pour interagir avec des modèles d'intelligence artificielle en local. L'accent est mis sur la confidentialité absolue des données, la flexibilité esthétique et une ergonomie fluide.

## ✨ Fonctionnalités

* **Architecture Mono-fichier :** Tout le projet (HTML5, CSS3, JavaScript) est encapsulé dans un fichier unique pour une portabilité totale et une exécution locale sans dépendances complexes.
* **Design Futuriste & Fluide :** Une interface épurée dotée d'une barre latérale dynamique qui s'adapte automatiquement au déploiement des options (configuration de clés API, gestion du scroll intelligent).
* **Entrée Vocale Intégrée :** Prise en charge du contrôle vocal avec un indicateur visuel dynamique (pulsation/clignotement) lors de l'écoute active.
* **Thèmes Personnalisables :** Commutation à la volée entre plusieurs ambiances visuelles optimisées :
    * *Original* : L'identité visuelle native.
    * *Épuré* : Un mode minimaliste et relaxant.
    * *Fennec Cyber* : Un mode sombre aux accents orange néon.
    * *Oasis Tech* : Une ambiance sombre teintée de vert émeraude reposant.
    * *Midnight Code* : Un noir absolu (OLED) à haut contraste.

## 🛠️ Technologies

* HTML5 / CSS3 (Variables CSS natives, Flexbox)
* JavaScript (Vanilla JS)
* Google Fonts (Orbitron)
